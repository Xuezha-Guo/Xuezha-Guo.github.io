一、IDEA配置相关问题

使用IDEA学习时遇到无法编译的问题，经过排查解决如下

问题如下
打开 IDEA ，点击 Remote Development -> SSH ，然后依次选择 IDE 版本和项目目录。打开之后编写代码并点击运行，运行失败报错：
{% highlight java linenos %} java: Cannot run program "/usr/local/jdk_1.8/bin/java" (in directory "/home/XXXX/.cache/JetBrains/RemoteDev-IU/_home_XXXX_Projects_demo/compile-server"): error=0, Failed to exec spawn helper: pid: 2483, exit value: 1 {% endhighlight %}
解决如下
取消勾选 Use ‘--release’ option for cross-compliation(java9 and later) 取消勾选 Use coompiler from module target JDK when possible 或指定Project bytecode version
